# DumpModule
Build with `-G "Visual Studio 16 2019`.

Requires administrative privileges.

### Example
```
$ ./DumpModule.exe
Enter PID: 14192
Enter module name: pso2.exe
Dumping pso2.exe...
pso2.exe
Module address: 4194304
Module size: 110485504
Opened process, dumping memory...
110485504 bytes read; saving...
Done!
```
